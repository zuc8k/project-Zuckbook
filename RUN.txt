ZuckBook - ONE COMMAND TO START
================================

DOCKER (Recommended):
  Windows: START.bat
  Linux/Mac: bash START.sh

XAMPP (Local):
  Windows: RUN_XAMPP.bat

Manual XAMPP:
  1. Start MySQL in XAMPP Control Panel
  2. Start Apache in XAMPP Control Panel
  3. cd socket-server && npm install && npm start
  4. Open: http://localhost:8080

Database:
  - Host: localhost
  - User: root
  - Password: 197520error
  - Database: zuckbook
  - Port: 3306 (XAMPP) or 3306 (Docker)

Access:
  http://localhost:8080

Services:
  - Apache: http://localhost:8080
  - MySQL: localhost:3306 or 3306
  - Socket.IO: http://localhost:3000

Stop:
  Docker: docker-compose down
  XAMPP: Stop services in Control Panel
